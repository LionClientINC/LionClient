A client made for the community. Custom base, but made partially with AI, using examples from certain versions of Raven (But not any raven versions made by Blowsy, so not for example, raven b4.) How to use: put the .jar into your mods folder and launch. The default ClickGUI bind is RSHIFT. Select modules to see the description. ClickGUI has 2 modes, Modern and Classic, Modern is a modern clickgui. Classic is a dropdown. To see module descriptions on classic you hover on the module for 2 seconds.
